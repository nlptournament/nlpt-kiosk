# Screen showing Plain Text

As the name suggests, this screen is meant to just show some Text on a black background.

# specific variables

| variable   | description                                                                                        |
| ---------- | -------------------------------------------------------------------------------------------------- |
| text       | thetext to be displayed, can be multiline                                                          |
| text_color | give the text some color (default `#f0f0f0`) Takes valied HTML color input e.g. `red` or `#FF0000` |
| text_size  | scale the text-size, takes values from 1 to 14. See the table below for details                    |
